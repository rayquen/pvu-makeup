# CHANGELOG

# 03/10/2021 Feature sketch-de-extension

Se plantea una arquitectura simple de la extension.

Se prepara la pagina principal de configuración y se añade una opcion para setear los procesos que modificaran PVU.

Se agrega una configuracion que se almacena en el storage sincronizado por usuario de google chrome.

Se incorpora un mecanismo simple para la obtencion de las cotizaciones y se calculan las diferentes conversiones entre las monedas mas importantes: PVU, ARS, USD y LE

Se construye el mecanismo para invocar codigo que modifica una pagina host invocada desde la pestaña activa. Esta invocación se realiza cada vez que el usuario activa la pestaña.
